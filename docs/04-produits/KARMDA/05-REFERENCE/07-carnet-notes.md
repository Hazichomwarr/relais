# KARMDA

# 07 - Carnet de notes

## Référence fonctionnelle

> Le module Carnet de notes permet de consulter les résultats consolidés des élèves d'une classe à partir des évaluations enregistrées.

---

# Objectif

Le Carnet de notes offre une vue d'ensemble des résultats académiques d'une classe.

Il rassemble les évaluations, les notes enregistrées et les moyennes afin de faciliter le suivi pédagogique des élèves.

Le Carnet de notes est un module de consultation.

Les évaluations et les notes sont créées dans le module **Notes**.

---

# Utilisateurs concernés

- Administrateur
- Personnel administratif
- Enseignant (selon les permissions)

---

# Permissions requises

Selon son rôle, un utilisateur peut :

- consulter un carnet de notes ;
- analyser les résultats d'une classe ;
- consulter les moyennes ;
- identifier les notes manquantes.

Les modifications des notes sont réalisées dans le module **Notes**.

---

# Vue d'ensemble

Le Carnet de notes est organisé par classe.

Après avoir sélectionné une classe, KARMDA présente :

- les évaluations disponibles ;
- les notes enregistrées ;
- les moyennes des élèves ;
- les notes manquantes ;
- les principaux indicateurs académiques.

---

# Fonctionnement

## 1. Sélectionner une classe

Choisir la classe dont les résultats doivent être consultés.

---

## 2. Afficher le carnet

KARMDA construit automatiquement une vue consolidée à partir des évaluations existantes.

---

## 3. Consulter les résultats

Pour chaque élève sont notamment affichés :

- les notes obtenues ;
- la moyenne ;
- le nombre de notes enregistrées ;
- les notes manquantes.

---

## 4. Analyser les indicateurs

Le carnet affiche également plusieurs indicateurs :

- nombre d'élèves ;
- nombre d'évaluations ;
- notes saisies ;
- notes manquantes ;
- moyenne générale de la classe.

Ces informations facilitent le suivi pédagogique.

---

# Règles métier

## Le carnet est généré automatiquement

Le Carnet de notes est construit à partir des évaluations existantes.

Aucune donnée n'est saisie directement dans ce module.

---

## Les moyennes sont calculées automatiquement

Les moyennes sont calculées à partir :

- des notes enregistrées ;
- des coefficients ;
- des règles de calcul définies par l'établissement.

---

## Les notes manquantes sont identifiées

Les évaluations sans résultat apparaissent comme des notes manquantes.

Elles permettent d'identifier rapidement les saisies incomplètes.

---

## Les absences et les notes excusées

Selon les règles de calcul de l'établissement, les absences, notes excusées et notes manquantes peuvent être exclues du calcul des moyennes.

---

# Cas d'utilisation

### Réunion pédagogique

Consulter les résultats d'une classe afin d'identifier les élèves nécessitant un accompagnement.

---

### Vérification

Contrôler que toutes les évaluations possèdent les notes attendues avant la production des bulletins.

---

# Erreurs fréquentes

- chercher à modifier une note depuis le Carnet de notes ;
- interpréter une moyenne sans tenir compte des notes manquantes ;
- oublier de sélectionner la bonne classe ;
- confondre le Carnet de notes avec le module Notes.

---

# Questions fréquentes

## Puis-je saisir une note depuis le Carnet de notes ?

Non.

La saisie est réalisée dans le module **Notes**.

---

## Pourquoi une moyenne n'apparaît-elle pas ?

Certaines évaluations ne possèdent pas encore de résultat ou les conditions de calcul ne sont pas réunies.

---

## Les moyennes sont-elles recalculées automatiquement ?

Oui.

Chaque nouvelle note ou correction met automatiquement à jour les résultats.

---

# Bonnes pratiques RELAIS

✔ Vérifier les notes manquantes avant chaque conseil de classe.

✔ Consulter régulièrement les indicateurs académiques.

✔ Utiliser le Carnet de notes pour analyser les performances de la classe.

✔ Effectuer toute correction dans le module **Notes**.

✔ Vérifier les moyennes avant la génération des bulletins.
