# KARMDA

# 03 - Les fonctionnalités

## Académie RELAIS

> Ce document fait partie du programme officiel de formation KARMDA.

**Objectif**

À la fin de cette lecture, vous serez capable de connaître les principales fonctionnalités de KARMDA, de comprendre leur utilité et d'expliquer la valeur qu'elles apportent à un établissement scolaire.

**Temps estimé**

25 minutes

**Prérequis**

02 - Vue d'ensemble

---

# Pourquoi ce chapitre est important

Une fonctionnalité n'a aucune valeur en elle-même.

Sa valeur dépend du problème qu'elle résout.

Chez RELAIS, nous ne présentons donc jamais une fonctionnalité avant d'avoir expliqué son utilité.

Lorsque vous présentez KARMDA, ne dites pas :

> "Nous avons un module de présences."

Préférez :

> "Vos enseignants n'auront plus besoin de remplir plusieurs registres chaque jour."

Le client achète une solution.

Jamais une fonctionnalité.

---

# Une journée avec KARMDA

Pour mieux comprendre KARMDA, suivons le déroulement d'une journée dans un établissement scolaire.

---

# Avant la rentrée

Avant même l'arrivée des élèves, KARMDA permet de préparer toute l'année scolaire.

Les équipes peuvent notamment :

- créer les années scolaires ;
- organiser les classes ;
- créer les sections ;
- enregistrer les matières ;
- créer les emplois du temps ;
- inscrire les élèves ;
- gérer les enseignants.

Toutes les informations sont prêtes avant le premier jour de cours.

---

# Arrivée des élèves

Les élèves arrivent à l'école.

Les présences sont enregistrées rapidement.

Les absences sont immédiatement visibles.

Selon les paramètres de l'établissement, les parents peuvent être informés.

### Valeur

L'école sait rapidement qui est présent.

Les familles sont rassurées.

---

# Pendant les cours

Les enseignants utilisent KARMDA pour gérer leur activité quotidienne.

Ils peuvent notamment :

- consulter leur emploi du temps ;
- accéder à leurs classes ;
- enregistrer les présences ;
- saisir les notes ;
- consulter les informations utiles concernant leurs élèves.

### Valeur

Moins de papier.

Moins de temps administratif.

Davantage de temps consacré à l'enseignement.

---

# Après une évaluation

Une fois une évaluation terminée, les notes sont enregistrées.

KARMDA calcule automatiquement les moyennes selon les règles définies par l'établissement.

Les résultats restent disponibles tout au long de l'année.

### Valeur

Réduction des erreurs.

Gain de temps.

Résultats fiables.

---

# Communication avec les familles

Les établissements peuvent partager différentes informations avec les parents.

Par exemple :

- annonces ;
- résultats ;
- présences ;
- devoirs ;
- événements.

### Valeur

Une meilleure communication.

Une relation école-famille renforcée.

---

# Administration

Tout au long de l'année, l'administration utilise KARMDA pour gérer :

- les élèves ;
- les enseignants ;
- les paiements ;
- les documents ;
- les inscriptions ;
- les statistiques.

### Valeur

Une gestion plus rapide.

Moins d'erreurs.

Toutes les informations au même endroit.

---

# Direction

La direction dispose d'une vision globale de son établissement.

Elle peut suivre :

- les effectifs ;
- les résultats scolaires ;
- les statistiques de fréquentation ;
- les performances académiques ;
- les indicateurs importants.

### Valeur

Décider avec des données fiables.

# Notre philosophie

Chaque fonctionnalité de KARMDA répond à une question simple.

> **Quel problème concret résout-elle ?**

Si nous ne pouvons pas répondre clairement à cette question,

alors cette fonctionnalité n'a probablement pas sa place dans le produit.

---

# Ce qu'il faut retenir

À l'issue de ce chapitre, retenez les idées suivantes :

- Les fonctionnalités de KARMDA accompagnent toute la vie de l'établissement.
- Elles sont pensées pour simplifier le travail des équipes.
- Elles fonctionnent ensemble dans une seule plateforme.
- Chaque fonctionnalité répond à un besoin réel.
- Une bonne démonstration présente toujours la valeur avant la fonctionnalité.
