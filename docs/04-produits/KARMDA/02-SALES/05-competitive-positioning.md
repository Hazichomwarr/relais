# KARMDA

# 06 - Positionnement concurrentiel

## Ventes RELAIS

> Ce document fait partie du programme officiel de formation commerciale KARMDA.

**Objectif**

À la fin de cette lecture, vous serez capable de présenter KARMDA face à d'autres solutions avec professionnalisme, objectivité et respect.

**Temps estimé**

25 minutes

**Prérequis**

01 - Script de démonstration

02 - Gérer les objections

03 - Tarification

04 - Études de cas

---

# Pourquoi ce document est important

Il est fréquent qu'un établissement compare plusieurs solutions avant de prendre une décision.

Cette démarche est normale.

Notre rôle n'est pas de convaincre un établissement que tous les autres logiciels sont mauvais.

Notre rôle est de l'aider à comprendre les différences afin qu'il puisse choisir la solution la plus adaptée à ses besoins.

---

# Notre philosophie

Chez RELAIS, nous ne parlons jamais contre un concurrent.

Nous parlons toujours pour notre client.

Nous respectons le travail des autres entreprises.

Nous préférons expliquer ce qui fait la force de KARMDA plutôt que chercher les faiblesses des autres.

Le respect est une valeur.

---

# Ce que nous comparons

Lorsque nous comparons une solution, nous parlons de critères.

Jamais de personnes.

Les critères les plus importants sont notamment :

- simplicité d'utilisation ;
- couverture fonctionnelle ;
- accompagnement ;
- support ;
- évolutivité ;
- sécurité ;
- coût global ;
- qualité de la formation ;
- facilité de déploiement.

---

# Les questions à poser

Avant de comparer, cherchez d'abord à comprendre.

Par exemple :

- Qu'utilisez-vous actuellement ?
- Qu'appréciez-vous dans cette solution ?
- Quelles difficultés rencontrez-vous ?
- Que souhaiteriez-vous améliorer ?

Ces réponses orienteront naturellement la conversation.

---

# Comment présenter KARMDA

Présentez KARMDA avec des faits.

Par exemple :

- plateforme centralisée ;
- accompagnement personnalisé ;
- formation des utilisateurs ;
- développement continu ;
- proximité avec les établissements ;
- documentation complète ;
- support réactif.

Ne faites jamais de promesse que vous ne pouvez pas démontrer.

---

# Si le client cite un concurrent

Écoutez.

Posez des questions.

Puis expliquez calmement comment KARMDA répond au besoin exprimé.

Par exemple :

> Je comprends. Cette solution répond effectivement à certains besoins. Si vous le souhaitez, regardons ensemble comment KARMDA traite ce même sujet afin que vous puissiez comparer les approches.

Vous accompagnez.

Vous ne débattez pas.

---

# Si le concurrent est moins cher

Ne cherchez pas immédiatement à justifier le prix.

Cherchez d'abord à comprendre.

Le client compare peut-être :

- les fonctionnalités ;
- l'accompagnement ;
- la formation ;
- le support ;
- la durée d'engagement ;
- les coûts cachés.

Le prix n'est qu'un critère parmi d'autres.

---

# Si le concurrent possède une fonctionnalité que KARMDA ne propose pas

Soyez honnête.

Vous pouvez répondre :

> Aujourd'hui, cette fonctionnalité n'est pas disponible dans KARMDA.

> En revanche, permettez-moi de vous montrer comment nous répondons à votre besoin avec les fonctionnalités actuellement disponibles.

Ne promettez jamais une fonctionnalité inexistante.

---

# Si KARMDA n'est pas la meilleure solution

Il est possible qu'un établissement ait des besoins auxquels KARMDA ne répond pas encore.

Dans ce cas :

Soyez transparent.

Expliquez les limites.

Proposez éventuellement de rester en contact pour les évolutions futures.

Nous préférons perdre une vente plutôt que perdre notre crédibilité.

---

# Ce qui différencie RELAIS

Nous souhaitons être reconnus pour :

- notre écoute ;
- notre accompagnement ;
- notre proximité ;
- notre transparence ;
- notre amélioration continue.

Notre différence ne repose pas uniquement sur le logiciel.

Elle repose aussi sur la manière dont nous accompagnons nos clients.

---

# Les erreurs à éviter

Ne jamais :

- critiquer un concurrent ;
- ridiculiser une autre solution ;
- diffuser des informations non vérifiées ;
- inventer des comparaisons ;
- promettre qu'un concurrent disparaîtra ;
- affirmer que KARMDA est parfait.

---

# Notre philosophie commerciale

La concurrence nous pousse à progresser.

Elle nous rappelle que chaque jour est une occasion d'améliorer KARMDA.

Nous ne cherchons pas à être les seuls.

Nous cherchons à être le meilleur partenaire possible pour les établissements qui nous font confiance.

---

# Ce qu'il faut retenir

À l'issue de ce chapitre, retenez les idées suivantes :

- Une comparaison doit rester professionnelle.
- Les faits valent mieux que les opinions.
- Les besoins du client sont plus importants que les concurrents.
- L'honnêteté renforce la confiance.
- Le respect fait partie de l'image de RELAIS.

---

2. Quels critères sont plus importants que le prix seul ?

3. Comment répondre lorsqu'un concurrent possède une fonctionnalité absente de KARMDA ?

4. Que faire si KARMDA ne répond pas au besoin d'un établissement ?

5. Quelle est la véritable différence entre RELAIS et ses concurrents ?

# Le Coach RELAIS

## Exercice 1

Votre collègue joue le rôle d'un directeur qui affirme :

> « Votre concurrent est meilleur. »

Votre objectif n'est pas de contredire cette affirmation.

Votre objectif est de comprendre pourquoi il pense cela.

---

## Exercice 2

Expliquez en deux minutes ce qui distingue KARMDA sans citer un seul concurrent.

---

## Exercice 3

Votre collègue vous présente une fonctionnalité que KARMDA ne possède pas.

Répondez avec honnêteté, puis recentrez la discussion sur la valeur apportée par KARMDA.

---

## La règle d'or

Avant chaque comparaison, posez-vous cette question :

> **« Est-ce que ce que je vais dire aide réellement le client à prendre une meilleure décision ? »**

Si la réponse est non, ne le dites pas.

---

## Phrase à retenir

> **« Chez RELAIS, nous ne gagnons pas en rabaissant les autres. Nous gagnons en démontrant, avec humilité, la valeur que nous apportons à nos clients. »**
