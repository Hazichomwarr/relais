# KARMDA

# Documentation Commerciale

Bienvenue dans la documentation commerciale officielle de KARMDA.

Cette documentation accompagne tous les collaborateurs RELAIS impliqués dans la présentation, la vente et le déploiement commercial de KARMDA.

Son objectif est de fournir un cadre commun afin que chaque établissement bénéficie d'une expérience cohérente, professionnelle et fidèle aux valeurs de RELAIS.

---

# Notre philosophie commerciale

Chez RELAIS, nous ne cherchons pas à vendre un logiciel.

Nous cherchons à comprendre les besoins des établissements scolaires et à les accompagner dans leur transformation.

Une vente réussie est une vente basée sur la confiance.

Notre objectif n'est pas de convaincre à tout prix.

Notre objectif est d'aider chaque établissement à prendre une décision éclairée.

---

# Comment utiliser cette documentation

Cette documentation n'a pas vocation à être mémorisée mot à mot.

Elle doit être comprise.

Avec l'expérience, chaque commercial développera son propre style de communication tout en respectant les principes de RELAIS.

Nous ne cherchons pas à produire des discours identiques.

Nous cherchons à offrir une expérience cohérente.

---

# Documents vivants

Cette documentation évolue en permanence.

Chaque démonstration, chaque objection, chaque retour client et chaque nouvelle idée peuvent contribuer à son amélioration.

Lorsqu'un collaborateur identifie une meilleure manière de faire, il est encouragé à la documenter afin que toute l'équipe puisse en bénéficier.

Chez RELAIS, une expérience individuelle devient une connaissance collective.

---

# Les principes commerciaux RELAIS

Avant chaque rencontre, souvenez-vous de ces principes.

- Comprendre avant de présenter.
- Écouter avant de répondre.
- La valeur avant le prix.
- Les faits avant les opinions.
- La confiance avant la vente.
- Les relations avant les transactions.
- L'honnêteté avant tout.

Ces principes guident chacune de nos interactions.

---

# Ce que nous attendons d'un commercial RELAIS

Un commercial RELAIS est avant tout un conseiller.

Il cherche à comprendre.

Il pose des questions.

Il écoute attentivement.

Il présente KARMDA avec honnêteté.

Il respecte les concurrents.

Il tient ses engagements.

Il accompagne le client bien au-delà de la signature du contrat.

---

# Une vente réussie

Chez RELAIS, une vente est considérée comme réussie lorsque :

- le client comprend clairement la valeur de KARMDA ;
- la solution répond réellement à ses besoins ;
- les engagements sont réalistes ;
- la relation de confiance est établie ;
- le partenariat peut s'inscrire dans la durée.

La signature d'un contrat n'est pas une fin.

C'est le début d'une collaboration.

---

# Contribution

Cette documentation appartient à toute l'équipe RELAIS.

Chaque collaborateur est invité à :

- signaler une information obsolète ;
- proposer une amélioration ;
- partager un retour d'expérience ;
- enrichir les études de cas ;
- compléter la FAQ ;
- documenter les nouvelles tendances du marché.

En améliorant cette documentation, vous améliorez la qualité de toutes les futures démonstrations.
