# Politique de remise

Les remises ne sont jamais automatiques.

Toute remise doit respecter la politique commerciale RELAIS.

| Niveau de remise | Autorisation requise   |
| ---------------- | ---------------------- |
| Jusqu'à X %      | Commercial             |
| De X % à Y %     | Responsable commercial |
| Au-delà de Y %   | Direction              |

Les seuils seront définis par la direction.

# Cas particuliers

Les situations suivantes nécessitent une validation de la direction :

- grands groupes scolaires ;
- établissements publics ;
- ONG ;
- appels d'offres ;
- partenariats stratégiques.

---

# Ce qu'un commercial ne peut jamais faire

Sans autorisation, un commercial ne peut pas :

- modifier un tarif ;
- promettre une remise ;
- offrir un module ;
- modifier les modalités de paiement ;
- engager RELAIS sur une fonctionnalité non prévue.

En cas de doute :

Demandez toujours validation.
