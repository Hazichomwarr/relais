# KARMDA

# 07 - Intelligence concurrentielle

## Ventes RELAIS

> **Document interne confidentiel.**
>
> Ce document est réservé aux collaborateurs RELAIS.
> Il ne doit jamais être communiqué à un client.

---

# Objectif

Centraliser les connaissances sur le marché de la gestion scolaire afin d'aider les équipes RELAIS à mieux comprendre leur environnement.

Ce document n'a pas pour objectif de critiquer les autres solutions.

Il permet de mieux connaître le marché afin de prendre de meilleures décisions.

---

# Pourquoi ce document est important

Aucun produit n'évolue dans le vide.

Chaque solution répond à une vision différente.

Comprendre les autres acteurs du marché permet :

- de mieux comprendre les attentes des établissements ;
- d'identifier les tendances du secteur ;
- d'améliorer continuellement KARMDA ;
- d'adapter notre positionnement.

Nous observons nos concurrents avec curiosité.

Jamais avec mépris.

---

# Notre philosophie

Chez RELAIS, nous considérons que la concurrence est une source d'apprentissage.

Chaque entreprise apporte une idée.

Chaque produit résout un problème d'une manière différente.

Notre objectif n'est pas de copier.

Notre objectif est de comprendre.

Puis d'innover.

## Forces

Documentez uniquement des faits observables.

Par exemple :

- interface intuitive ;
- documentation complète ;
- application mobile ;
- forte présence sur le marché.

---

## Limites observées

Toujours rester factuel.

Ne jamais utiliser de termes dévalorisants.

Exemples :

- certaines fonctionnalités nécessitent plusieurs manipulations ;
- absence d'un portail parents ;
- peu de personnalisation.

Les limites doivent être vérifiables.

---

# Les tendances du marché

Ce document doit également suivre les grandes évolutions du secteur.

Par exemple :

- intelligence artificielle ;
- applications mobiles ;
- signatures électroniques ;
- communication WhatsApp ;
- paiement Mobile Money ;
- tableaux de bord avancés.

Les tendances d'aujourd'hui peuvent devenir les attentes de demain.

---

# Les demandes récurrentes des établissements

Recensez toutes les demandes exprimées par les prospects.

Par exemple :

- portail parents ;
- notifications SMS ;
- paiement des frais en ligne ;
- bulletins numériques ;
- application mobile.

Si plusieurs établissements demandent la même chose, cette information devient précieuse pour la feuille de route produit.

# Ce que nous ne faisons jamais

Nous ne :

- copions pas une fonctionnalité sans comprendre son intérêt ;
- diffusons pas de rumeurs ;
- utilisons pas d'informations confidentielles ;
- espionnons pas nos concurrents.

Notre veille est réalisée uniquement à partir d'informations publiques et de retours légitimes de nos clients.

---

# Comment utiliser ce document

Avant une rencontre importante :

- consultez les informations disponibles ;
- identifiez les solutions déjà utilisées par le client ;
- préparez les points de comparaison utiles.

Après une rencontre :

- ajoutez les nouvelles informations apprises ;
- vérifiez les sources ;
- mettez à jour les tendances observées.

Ce document est vivant.

Il évolue avec le marché.

---

# Les sources d'information

Les informations doivent provenir de sources fiables.

Par exemple :

- sites officiels ;
- documentations publiques ;
- démonstrations officielles ;
- salons professionnels ;
- publications de l'entreprise ;
- retours documentés des clients.

Chaque information importante doit pouvoir être vérifiée.

# Ce qu'il faut retenir

- Comprendre le marché permet de prendre de meilleures décisions.
- La concurrence est une source d'apprentissage.
- Les informations doivent toujours être factuelles et vérifiables.
- Une bonne veille nourrit l'amélioration continue de KARMDA.
- Nous observons les autres pour progresser, jamais pour les dénigrer.
