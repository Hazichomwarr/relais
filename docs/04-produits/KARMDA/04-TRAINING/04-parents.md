# KARMDA

# 04 - Formation des parents

## Formation RELAIS

> Ce document présente le programme officiel de formation des parents utilisant KARMDA.

**Objectif**

À la fin de cette formation, chaque parent sera capable de suivre la scolarité de son enfant, consulter les informations importantes, comprendre les résultats scolaires et communiquer efficacement avec l'établissement grâce à KARMDA.

**Durée indicative**

45 à 60 minutes

**Public concerné**

- Parents
- Tuteurs légaux
- Responsables d'élèves

**Prérequis**

- Posséder un compte KARMDA actif
- Être rattaché à au moins un élève

---

# Pourquoi cette formation est importante

L'éducation d'un enfant est une responsabilité partagée entre la famille et l'établissement.

Lorsque les parents disposent d'informations fiables et à jour, ils peuvent accompagner leur enfant plus efficacement.

KARMDA permet aux parents de suivre la vie scolaire sans attendre les réunions de fin de trimestre.

Vous pouvez notamment consulter :

- les présences ;
- les absences ;
- les résultats scolaires ;
- les bulletins ;
- les communications de l'établissement ;
- les informations administratives ;
- les frais scolaires.

L'objectif n'est pas de surveiller votre enfant en permanence.

L'objectif est de mieux l'accompagner.

---

# Notre philosophie

KARMDA ne remplace pas les échanges entre les parents et l'école.

Il les facilite.

Les informations disponibles dans KARMDA permettent d'agir plus tôt lorsqu'une difficulté apparaît.

> **Un parent ne peut pas être présent à chaque cours. KARMDA lui permet de rester présent dans le parcours scolaire de son enfant.**

---

# Le rôle du parent

Le parent consulte les informations concernant son enfant.

Il peut notamment :

- suivre les présences ;
- consulter les résultats ;
- lire les communications de l'école ;
- suivre les frais scolaires ;
- télécharger les bulletins lorsque ceux-ci sont publiés.

Le parent ne modifie pas :

- les notes ;
- les présences ;
- les appréciations ;
- les informations académiques.

En cas d'erreur, il contacte l'établissement.

---

# Résultats attendus

À la fin de cette formation, vous devez pouvoir :

- vous connecter ;
- consulter votre tableau de bord ;
- suivre les présences ;
- consulter les résultats ;
- comprendre un bulletin ;
- consulter les frais scolaires ;
- recevoir les communications ;
- protéger votre compte ;
- demander de l'aide.

# 1. Première connexion

Chaque parent reçoit un compte personnel.

Utilisez uniquement votre propre compte.

Ne partagez jamais :

- votre mot de passe ;
- votre code de connexion ;
- vos informations personnelles.

Votre compte contient des informations confidentielles concernant votre enfant.

---

# 2. Sécuriser son compte

Pour protéger votre compte :

- choisissez un mot de passe sécurisé ;
- ne communiquez jamais votre mot de passe ;
- déconnectez-vous sur un appareil partagé ;
- contactez l'établissement en cas de doute.

La sécurité de votre compte protège également les informations de votre enfant.

---

# 3. Comprendre votre tableau de bord

Le tableau de bord rassemble les informations essentielles.

Vous pouvez y retrouver :

- vos enfants ;
- les dernières absences ;
- les derniers résultats ;
- les prochaines informations importantes ;
- les annonces de l'établissement.

Prenez quelques minutes chaque semaine pour consulter votre tableau de bord.

---

# 4. Suivre les présences

Les présences permettent de vérifier la participation de votre enfant aux cours.

Soyez attentif :

- aux absences répétées ;
- aux retards fréquents ;
- aux absences non justifiées.

Une absence isolée n'est pas toujours inquiétante.

En revanche, des absences répétées doivent conduire à un échange avec votre enfant ou avec l'établissement.

---

# 5. Consulter les résultats

Les résultats permettent de suivre les progrès de votre enfant.

Ils ne servent pas uniquement à constater une bonne ou une mauvaise note.

Ils permettent également :

- d'identifier les matières difficiles ;
- de reconnaître les progrès ;
- d'encourager les efforts.

Les résultats doivent être un outil de dialogue, pas une source de pression.

---

# Comprendre une baisse de résultats

Une baisse ponctuelle ne signifie pas forcément un problème durable.

Avant de tirer une conclusion :

- échangez avec votre enfant ;
- consultez les appréciations des enseignants ;
- contactez l'établissement si nécessaire.

L'objectif est de comprendre avant de juger.

---

# 6. Comprendre les bulletins

Le bulletin rassemble plusieurs informations importantes :

- les notes ;
- les moyennes ;
- les appréciations ;
- les observations.

Prenez le temps de lire l'ensemble du bulletin.

Les commentaires des enseignants sont souvent aussi importants que les notes.

---

# 7. Suivre les frais scolaires

Selon les choix de votre établissement, KARMDA peut permettre de consulter :

- les paiements effectués ;
- les soldes restants ;
- les échéances à venir ;
- les reçus.

En cas d'incohérence, contactez directement l'administration.

---

# 8. Recevoir les communications

L'établissement peut publier :

- des annonces ;
- des convocations ;
- des réunions ;
- des événements ;
- des rappels.

Consultez régulièrement ces communications afin de rester informé.

---

# 9. Signaler une information incorrecte

Si une information semble erronée :

Ne modifiez rien.

Contactez l'établissement.

Expliquez précisément :

- l'élève concerné ;
- l'information observée ;
- la raison de votre demande.

L'administration vérifiera les informations avant toute correction.

---

# 10. Confidentialité

Les informations de votre enfant sont personnelles.

Ne partagez pas :

- les bulletins ;
- les notes ;
- les captures d'écran ;
- les informations d'autres élèves.

Respecter la confidentialité protège tous les enfants.

---

# Comment accompagner votre enfant grâce à KARMDA

Utilisez les informations pour :

- encourager les efforts ;
- valoriser les progrès ;
- détecter rapidement les difficultés ;
- préparer les échanges avec les enseignants.

Évitez d'utiliser KARMDA uniquement lors des mauvaises notes.

Un suivi régulier est plus efficace qu'une réaction tardive.

# Erreurs fréquentes

- attendre la fin du trimestre pour consulter les résultats ;
- partager son compte avec une autre personne ;
- ignorer les absences répétées ;
- consulter uniquement les notes sans lire les appréciations ;
- comparer systématiquement son enfant aux autres.

---

# Bonnes pratiques RELAIS

✔ Consulter régulièrement KARMDA.

✔ Encourager les progrès de votre enfant.

✔ Dialoguer avec les enseignants.

✔ Signaler rapidement une anomalie.

✔ Protéger votre compte.

✔ Respecter la confidentialité.

✔ Utiliser KARMDA comme un outil d'accompagnement.

---

# Ce qu'il faut retenir

- KARMDA permet de suivre la scolarité de votre enfant.
- Les informations doivent servir à accompagner, pas à sanctionner.
- Les communications de l'école sont importantes.
- Les bulletins doivent être lus dans leur ensemble.
- Les absences répétées doivent être prises au sérieux.
- En cas de doute, contactez l'établissement.

# Question de réflexion

Depuis plusieurs semaines, vous remarquez que votre enfant accumule des retards et que ses résultats baissent progressivement en mathématiques. Les appréciations de l'enseignant indiquent également un manque de participation en classe.

Expliquez :

- quelles informations de KARMDA vous ont permis de détecter cette situation ;
- comment vous engageriez une discussion constructive avec votre enfant ;
- à quel moment vous contacteriez l'établissement ;
- pourquoi il est préférable d'agir dès les premiers signes plutôt que d'attendre le bulletin trimestriel ;
- comment une collaboration entre les parents et l'école peut favoriser la réussite de l'élève.
